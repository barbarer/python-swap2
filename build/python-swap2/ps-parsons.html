Swapping the Values of Two variables¶

When you swap the values of two variables such as x and y, x will have the initial value in y and y will have the initial value in x. If x’s initial value was 6 and y’s was 2 then after the swap x will have a value of 2 and y will have a value of 6.

Run the code below? Does it correctly swap the values?
